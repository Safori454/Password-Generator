# Password Generator Web App

A secure and customizable password generator built with "HTML, CSS, and JavaScript". Generate strong passwords with real-time strength estimation and multiple themes.

---

## Features

- "Set password length" (4â€“64 characters)  
- "Choose character types": lowercase, uppercase, numbers, symbols  
- "Password strength estimation" with entropy in bits  
- "Strength meter": Weak, Medium, Strong  
- "Copy to clipboard" with feedback  
- "Themes": Glass, Clean, Dark, Neon  
- "Dark mode toggle" synced with theme  
- "Keyboard shortcuts": `Enter` generates password, `Ctrl+G` / `Cmd+G` generates password  
- "Accessibility support" for keyboard navigation  

---

## Usage

1. "Open "https://safori454.github.io/Password-Generator/" in a browser"  
2. "Select character types"  
3. "Set password length"  
4. "Click "Generate""  
5. "Copy password" with the "Copy" button  
6. "Check strength" via meter and entropy display  
7. "Switch themes" or toggle dark mode
